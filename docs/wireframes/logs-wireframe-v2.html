Feature Checklist

All capabilities from research incorporated into the design

Virtualized scrolling

Smooth rendering of thousands of events without browser crash. Event list virtualized with windowing.

Timeline axis

Visual time ruler at the top shows event density. Each event positioned by timestamp on the axis.

Full JSON payload

Click any event to open a slide-in detail drawer with the complete input/output JSON, syntax-highlighted.

Search sessions

Real search input filtering sessions by ID, prompt text, or status. Debounced to prevent thrashing.

Filter by event type

Toggle chips for tool_call, tool_result, reasoning, error, delegate, mode, token. Multi-select support.

Per-event metrics

Duration, token count, and cost shown inline on every event. Color-coded for quick scanning.

Subagent tree view

Nested events indented with a left border. Child agent events visually linked to parent delegate_task.

Saved presets

Quick filter presets: "Errors only", "Last 5 min", "High cost". Create and save custom presets.

Focus / fullscreen

Collapse sidebar and chrome to let events fill the entire viewport. Maximizes reading density.

Live tailing

Auto-scroll to latest events during active agent execution. Visual indicator shows live status.

Related events linking

Clicking a reasoning event shows which tool_call and tool_result followed. Bidirectional navigation.

Cost & token tracking

Footer bar shows cumulative session cost and token usage. Per-event breakdown in detail drawer.

Clickable URLs

HTTP URLs in tool_call arguments and tool results are rendered as interactive links.

Color-coded types

Each event type has a distinct color: blue (tool_call), green (result), amber (reasoning), red (error), purple (delegate), orange (mode).

Status badges

Session list shows running/completed/failed/pending status dots with pulse animation for live sessions.
